## 주요 안건 및 논의사항

### 1. 팀 목표 설정
**핵심 목표**:
- 고급 과제를 포트폴리오로 적극 활용
- 과제 완료 후 포트폴리오 수준의 결과물 도출
- 1단계 구현 목표: 텍스트만을 입력하여 광고 컨텐츠(이미지, 문구 등) 생성
- 2단계 구현 목표: 추가 기능을 통한 품질 개선

### 2. 핵심 논의사항
**주요 이슈**:
- 프로젝트 방향성에 대한 생각 공유 후 최종 결정 내용
목적: 온라인 마케팅에 어려움을 겪는 소상공인이 손쉽게 이용할 수 있는 광고 생성 서비스

**구현 기능:**
1. 상담 챗봇 - 광고나 디자인에 대한 고민과 걱정을 해소할 수 있도록 도와줍니다
2. 대화 기반 광고 생성 - 자연스러운 대화를 통해 원하는 광고를 만들 수 있습니다
3. 플랫폼 최적화 - 각 플랫폼에 맞는 형식으로 광고가 생성됩니다
4. 챗 히스토리 저장 - 과거 대화 기록을 바탕으로 일관된 스타일의 광고를 생성할 수 있습니다

**워크플로우:**
1. 광고 형태 선택 (문구 / 이미지 / 영상)
2. 편집 또는 생성 선택 (편집: 기존 이미지 활용 / 생성: 새로 만들기)
3. 플랫폼 선택 (인스타그램 / 네이버 / 배달의민족 / 자유형식)
4. 광고 생성

**I2I의 방향성:**
목적: 사용자가 이미지를 입력했을 때 광고 생성에 반영
- 배경만 제거해서 이미지에 합성

**역할 변경:**
- 배현석: 텍스트, 프롬프트 생성 -> 챗봇 구성
- 이현석: 프롬프트 생성 추가
- 신승목: 텍스트 생성 추가
- 해당 내용에 맞게 generation 폴더 재구성 수행 완료
