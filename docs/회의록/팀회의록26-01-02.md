**주요 이슈**:

- (백엔드)
    - PostgreSQL은 서버에 데이터 저장하는 것을 상정하고 있기 때문에 서버 구조 필요
    - 외부 서버에 저장하는 것은 비용 청구 우려

- (프로젝트 매니저)
    - 개인 체크리스트 작성 및 전달 필요

**팀 결정사항**:
- (프론트)
    - Svelte를 이용하여 프론트엔드 UI 구성
- (백엔드)
    - GCP VM에 db를 저장하는 형태로 구성, 기초적인 배포 환경이 갖추어지면 db 저장에 대한 테스트 가능할 것
    - 기초적인 배포 환경 구성 필요(배포 담당: 신승목)
- (매니저)
