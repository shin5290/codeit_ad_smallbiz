## 260106
### (브리핑 내용)
**백엔드**
- JWT(Json Web Token)을 이용한 사용자 인증 방식 작업 중

**이미지 생성**
- SDXL 활용한 이미지 생성은 현재 완료, 베이스라인 설정 고민중
- 생성된 프롬프트 대로 이미지가 잘 생성되는지 확인 필요
- (예정) 이미지 입력 후에 레퍼런스 이미지를 참고해서 로고나 제품 이미지를 넣을 수 있는 툴들 확인 및 테스트

**텍스트 생성**
- 광고 문구(후처리 포함) 생성 구조 완료
- 기능 통합을 위한 코드 준비

**인프라**
- 이미지 생성과 텍스트 생성을 통합하는 테스트 환경 우선 구성 필요

### (피드백 내용)
- 서비스 평가 기준 확보
- 다양한 케이스로 테스트해서 케이스마다 차이가 있는지 확인
