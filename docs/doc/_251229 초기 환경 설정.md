# 사용자 정보 등록 및 계정 생성: GCP SSH 이용
- 정보 등록: jupyterhub_config.py
- 관리자 권한 추가
sudo -i
nano /etc/jupyterhub/jupyterhub_config.py
jupyterhub_config.py에서 특정 사용자 추가
c.Authenticator.allowed_users = {'사용자아이디'}
설정 후 Ctrl+O enter Ctrl+X로 저장 후 종료

- 계정 생성
useradd -m -s /bin/bash userid
- 비밀번호 설정
passwd userid
- 비밀번호 재설정
sudo passwd userid
- 현재 리눅스에 생성된 계정 확인
ls /home
- 새 인증 정보를 인식할 수 있도록 서비스 재시작
sudo systemctl restart jupyterhub

# Jupyterlab 접속 후 GCP 설정 시 설치한 torch 등을 가상환경에 가져오는 방법
- GCP SSH에서 실행, userid에 sudo 권한 부여
sudo usermod -aG sudo userid
- sudo 권한 있는지 확인
groups userid
- jupyterhub 재시작
systemctl restart jupyterhub

- jupyterlab (외부IP:8000)접속해서 계정이름과 비밀번호로 접속
- 터미널에서 실행, jupyterlab 계정의 비밀번호 필요
sudo python3 -m venv --system-site-packages /opt/jhub-venv
- 가상환경 활성화 (python3.10)
source /opt/jhub-venv/bin/activate

- apt로 설치한 라이브러리를 가상환경이 인식하지 못한다면 jupyterhub_config.py
c.Spawner.env_keep = ['PATH', 'PYTHONPATH', 'LD_LIBRARY_PATH']
