## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:
```
1. 스타일 자동 감지 시스템
   - style_router.py: 사용자 입력에서 키워드 기반으로 스타일 자동 감지
   - "애니메이션", "일러스트" → anime
   - "초사실적", "포토리얼리스틱" → ultra_realistic
   - "아기자기한", "귀여운" → semi_realistic

   - 각 스타일별로 고유한 YAML 설정 파일 생성
   - base_prompt, quality_tags, negative_keywords 등 스타일별 맞춤 설정
   - 유지보수성 향상 (스타일 추가 시 YAML만 추가)

3. 프롬프트 생성 고도화
   - 업종 + 스타일 + 사용자 입력 → 통합 프롬프트 생성
   - GPT를 통한 자연어 처리로 더 자연스러운 프롬프트 생성
```
