## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:
```
1. 해시 생성 버그
   - hashlib.sha256()은 bytes만 받음, PIL.Image 직접 전달 불가
   - io.BytesIO()로 메모리에 JPEG 저장 후 bytes 추출하여 해시 생성

2. JPEG 저장
   - .jpg 확장자만으로는 포맷 결정 안 됨
   - image.save(path, format='JPEG') 명시 필요
   - SDXL은 RGB 모드 출력이므로 JPEG 변환 문제 없음

3. 후처리 노드 분리 설계
   - BackgroundRemovalNode: rembg 기반 배경 제거
   - ImageOverlayNode: 9-point 앵커 기반 이미지 합성
   - TextOverlayNode: 텍스트 오버레이
```
