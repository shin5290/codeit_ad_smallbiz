## [3] 오늘 작업 완료도 체크 (하나만 체크)
- [ ] 0% (시작 못함)
- [ ] 25% (시작은 했지만 진척 없음)  
- [V] 50% (진행 중, 절반 이하)
- [ ] 75% (거의 완료됨)
- [ ] 100% (완료 및 점검까지 완료)

**간단한 근거**:
```
50% : 생각보다 생성 쪽에서 기본적인 생성이 끝나서 서로 연결될 수 있는 통합 테스트 환경 구성을 먼저 생각했었는데, 체크리스트 기준으로 먼저 GCP VM 내에서 가상환경 사용 시 사용하는 라이브러리가 중복으로 설치되지 않게 하기 위한 설정 내용을 조사하였고, 우선 requirements.txt 통합 및 가상환경 구성이 필요할 것으로 생각하였습니다.
```
