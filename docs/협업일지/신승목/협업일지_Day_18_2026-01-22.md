## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:  
```
현재 A/B 테스트를 18개 세부 업종에 대해서 10개씩, E등급만 각각 5개씩 샘플 텍스트를 생성하여 총 170개의 샘플 텍스트를 만들어서 진행하였습니다.
이미지 생성 프롬프트 평가 결과는 170:0으로 나왔습니다.
광고 문구는 146:5(무승부):19로 나왔습니다.
이미지 생성 프롬프트로 실제 이미지가 어떤 차이를 보이는 지 확인할 수 있다면 좋겠지만, 그렇지 못해서 평가 결과의 활용도가 제한적입니다.
광고 문구는 업종을 제대로 인식하지 못해서 완전히 엉뚱한 광고 문구를 생성한 경우가 있어서 업종 인식 단계에서 보완이 필요할 것으로 생각합니다.
이미지 생성 프롬프트도 업종 인식에서 문제가 있기 때문에 보완할 필요가 있을 것으로 생각합니다.

logging.py 개선을 위한 pull request 내용이 현재 테스트 상황과는 맞지 않는다는 문제가 있어서 진한 글씨로 나오게 하고, 개수 제한은 없애고, 용량 제한이랑 기간 제한(2주)으로 바꿔서 다시 pull request 내용을 준비할 필요가 있습니다.
```
