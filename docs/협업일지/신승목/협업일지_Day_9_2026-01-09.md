```

    `BASE_DIR = Path(__file__).parent.parent`
항목으로 하면 통일될 것 같은데, 처음 디렉토리 구성할 때 이미지 생성과 텍스트 생성을 src.generation 아래에 image_generation, text_generation으로 해둬서 현재 구성한 코드에서도 모듈 임포트 시 내용이 길어서 상대경로로 지정하는 것이 문제라고 판단하였다.
```
