## 1. 개요

### 1.1 배경

소상공인시장진흥공단의 247개 세부 업종을 효과적으로 분류하고, 각 업종 특성에 최적화된 AI 기반 광고 생성 전략을 수립하기 위해 본 분류 체계를 마련하였습니다.

### 1.2 분류 체계 개요

- **1차 분류**: 6개 등급 (S, A, B, C, D, E)
- **2차 분류**: 18개 하위 그룹
- **총 업종 수**: 247개

### 1.3 분류 기준 (Selection Logic)

| 순위 | 분류 기준 | 설명 |
|------|----------|------|
| 1 | **소비 접점 (B2C 친화도)** | 일반 소비자가 일상에서 얼마나 자주 검색하고 방문하는지 |
| 2 | **비주얼 의존도** | 이미지 한 장이 구매 결정에 미치는 영향력 |
| 3 | **경쟁 강도** | 주변 경쟁 업체의 밀집도에 따른 차별화 필요성 |

## 6. 결론 및 활용 방안

### 6.1 분류 체계 요약

본 보고서는 소상공인시장진흥공단의 **247개 업종**을 **6개 등급, 18개 하위 그룹**으로 체계적으로 분류하였습니다.

| 등급 | 하위 그룹 수 | 업종 수 | 핵심 전략 |
|------|-------------|---------|----------|
| S등급 | 4개 그룹 | 약 70개 | 맛의 시각화, 감성 자극 |
| A등급 | 4개 그룹 | 약 50개 | 변화/결과 강조, 스타일 제안 |
| B등급 | 3개 그룹 | 약 45개 | 공간의 미학, 체험 유도 |
| C등급 | 3개 그룹 | 약 50개 | 전문성/신뢰 기반 솔루션 |
| D등급 | 2개 그룹 | 약 30개 | 정보/혜택 중심, 실용성 |
| E등급 | 2개 그룹 | 약 4개 | 접근성/편의성, 정직함 |

### 6.2 활용 방안

#### 6.2.1 AI 광고 생성 시스템 구축

1. **업종 자동 감지**: 사용자 입력에서 키워드를 분석하여 18개 그룹 중 적합한 그룹으로 자동 분류
2. **프롬프트 자동 생성**: 그룹별 프롬프트 템플릿을 적용하여 이미지/문구 생성
3. **품질 관리**: 네거티브 프롬프트를 시스템 레벨에서 자동 적용

#### 6.2.2 UI/UX 설계 권장사항

1. **단계적 선택**: 247개 업종을 모두 나열하지 말고, 6개 등급 → 18개 그룹 → 세부 업종 순으로 선택하도록 설계
2. **시각적 아이콘**: 각 등급/그룹을 대표하는 아이콘으로 직관적인 선택 지원
3. **추천 기능**: 사용자의 이전 선택이나 입력 키워드를 기반으로 적합한 그룹 추천

#### 6.2.3 성과 측정 지표

| 등급 | 핵심 KPI | 목표 수치 |
|------|---------|----------|
| S등급 | CTR, 주문 전환율 | CTR 3-5%, 전환율 2-4% |
| A등급 | 예약율, 신규 고객 비율 | 예약율 5-10% |
| B등급 | 방문 예약율, 전화 문의 | 예약율 8-12% |
| C등급 | 상담 신청률, 리드 품질 | 상담율 10-15% |
| D등급 | 방문율, 전화 문의 | 문의율 5-8% |
| E등급 | 방문율, 이용률 | 방문율 3-5% |

### 6.3 향후 개선 방향

1. **피드백 루프 구축**: 광고 성과 데이터를 기반으로 프롬프트 최적화
2. **계절성 대응**: 고계절성 업종(꽃집, 여행사 등)에 대한 시즌별 프롬프트 변형
3. **공공데이터 연동**: 업력, 고객 만족도 등 객관적 지표를 카피에 자동 반영
4. **A/B 테스트**: 그룹별 프롬프트 변형을 통한 지속적인 성과 개선
